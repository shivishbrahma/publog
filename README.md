# Publog

Developer Blog of Shivishbrahma

## Themes

-   Background: #161925
-   Text: #f8f8f2
-   Font: Libre Baskerville
-   Language: English / Bengali

## Prequisites

-   Ruby 3.3

-   I18nNumber
-   I18nDate
-   I18nTime

    -   i18n_date
